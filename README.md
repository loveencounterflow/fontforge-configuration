# FontForge Configuration Files

```
fontforge-configuration .
-    ├── .config
-    │   └── fontforge
-    │       └── hotkeys
-    ├── fontforge
-    │   └── share
-    │       ├── default
-    │       └── default-n
-    ├── README.md
-    └── used-single-letter-hotkeys
```

* in `~/.config/fontforge`: `ln -s /path/to/repos/fontforge-configuration/.config/fontforge/hotkeys`
* will be picked up when FontForge starts
* no guarantee that menu entries will reflect change
